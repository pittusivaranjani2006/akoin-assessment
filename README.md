# Institute of Digital Risk (IDR) Homepage

This project contains a responsive homepage design for the Institute of Digital Risk (IDR), an industry-led training and deployment institute focused on digital, cyber, and AI risk.

## Features
- Responsive design (desktop + mobile)
- Sticky navigation with smooth scrolling
- Hero section with CTA
- Service pillars section
- Community section
- Contact form

## Logo Design
The IDR logo uses a cube-inspired geometric icon to represent structure, stability, and resilience in digital risk management.  
The orange color reflects innovation and technology leadership, while black and white provide a clean professional contrast.  
The cube form symbolizes structured risk frameworks and digital security systems.

## Technologies Used
- HTML5
- CSS3 (Flexbox / Grid)
- Vanilla JavaScript
